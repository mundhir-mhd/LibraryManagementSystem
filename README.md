# Library Management System

A C# console application to manage library books.

## Features
- Add new books
- View all books
- Search books by title
- Delete books

## How to Run
1. Open project in Visual Studio
2. Press F5 to run
3. Use menu options 1-5

## Technologies Used
- C#
- .NET Framework
- Collections (List)
